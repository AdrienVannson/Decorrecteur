À propos

Décorrecteur orthographique est un outil permettant d'ajouter de nombreuses fautes d'orthographe à un texte initialement correct.
Il est entièrement open-source (licence MIT): https://github.com/AdrienVannson/Decorrecteur
Pour fonctionner, Décorrecteur orthographique utilise la base de données Lexique383.
